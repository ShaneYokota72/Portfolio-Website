Maze Input

To begin with, there are mainly 4 characters that you are able to input.
You can input
'.', which represents an Enpty Space.
'#', which represents a Wall/Fence.
'S', which represents the Start Location.
'F', which represents the Finish Location.
Character 'S' and 'F' are REQUIRED for the program to run. Rest of the places may be filled with '.' or '#'

The Result will be printed in Console with a new character '*' representing the fastest path. To access the Console
1. Right click on the screen
2. Select Inspect
3. Click Console on the top bar.
